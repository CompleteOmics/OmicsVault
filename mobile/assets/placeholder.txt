Place the following image assets in this directory:

1. icon.png - App icon (1024x1024 px)
2. splash.png - Splash screen image (1284x2778 px for iPhone 14 Pro Max)
3. adaptive-icon.png - Android adaptive icon foreground (1024x1024 px)
4. favicon.png - Web favicon (48x48 px)

Recommended:
- Use the OmicsVault logo/icon with a blue (#0C88F1) background
- Keep icons simple and recognizable at small sizes
- Use transparent PNG format for adaptive-icon

You can generate these using:
- Figma or Adobe XD for design
- https://appicon.co/ for generating all required sizes
- https://makeappicon.com/ as an alternative
